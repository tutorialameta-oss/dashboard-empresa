# dashboard-empresa

## Archivos principales

- `index.html`: versión principal de la app.
- `index_corregido.txt`: copia en texto plano del `index.html` para descarga o pegado rápido.

## Uso rápido

1. Abre `index.html` en tu navegador.
2. Si deseas compartir una versión editable por texto, usa `index_corregido.txt`.

## Notas

- El dashboard es estático y no requiere instalación de dependencias.
- Para evitar problemas de caché, la página incluye encabezados meta para desactivar almacenamiento local del navegador.
